# Unity Solar System

Unity 3D 태양계 구현 프로젝트입니다.

## Project Overview

- 태양을 중심으로 8대 행성 구현
- 각 행성의 자전 및 공전 구현
- 일부 행성의 위성 구현
- Unity C# 스크립트를 사용한 회전 시스템 제작

## Features

### Rotation (Self Rotation)
- 태양, 행성, 위성이 각각 자전

### Revolution (Orbit)
- 행성은 태양을 중심으로 공전
- 위성은 부모 행성을 중심으로 공전
- 달은 부모 행성을 중심으로 공전 및 자전

### Unity Features Used
- Transform
- Rotate()
- RotateAround()
- Local / World Transform
- Hierarchy Structure
